Q2 Lesson 1: Introduction to CSS and Layers of a Webpage


What is CSS?

CSS stands for Cascading Style Sheets.
It defines how HTML elements are displayed on screen or other media.
Separates content (HTML) from design (CSS).
Enables consistent styling across multiple web pages using external stylesheets.

Why Use CSS?

Controls layout, design, and responsiveness.
Saves time by applying one stylesheet to many pages.
Enhances accessibility and visual appeal.
Improves site maintenance and consistency.

Layers of a Webpage:

Content Layer (HTML) – Structure and content (text, images, videos, etc.).
Presentation Layer (CSS) – Visual styling (colors, layout, fonts).
Behavior Layer (JavaScript) – Interactivity and user engagement.

Content Layer Essentials:

Clear, structured, and accessible text.
Use of alternative text for media.
Avoid relying solely on visuals for meaning.
Ensure clarity for all users, including those with disabilities.

Presentation Layer Essentials:

Ensure readability (zoomable text, contrast).
Consistent navigation and layout.
Highlight interactive elements clearly.

Behavior Layer Essentials:

Ensure functionality across input devices.
Simplify user experience by minimizing unnecessary options.

Benefits of CSS:

Efficiency as you can write the code once and apply it everywhere
Easy Maintenance Update styles across pages quickly.
Better Design: More control over layout and aesthetics.
Performance: Faster loading and reduced redundancy.

Capabilities of CSS:

Update styles site-wide.
Precisely position and layer elements.
Create custom tags for specific styling needs.

Prerequisite:

Basic knowledge of HTML or XHTML is required to learn CSS.
