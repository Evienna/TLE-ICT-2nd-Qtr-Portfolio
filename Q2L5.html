What Was Learned

I learned that CSS Selectors determine which HTML elements to style and they should be chosen 
based on the specificity needed. Types of selectors include:

Element/Tag Selector: targets all elements of a type.
ID Selector (#id): styles a unique element.
Class Selector (.class): styles multiple elements.
Universal Selector (*): targets all elements.
Group Selector (h1, p): applies the same style to multiple elements.
Attribute Selector (p[class]): targets elements with specific attributes.
