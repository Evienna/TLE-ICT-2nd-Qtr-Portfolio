What Was Learned

I learned that the display property controls how elements are rendered (block vs. inline). 
Block elements (e.g., <p>, <div>) stack vertically while inline elements (e.g., 
<span>, <a>) flow horizontally. Div is used for block-level grouping (paragraph/s 
or section/s) and span is for inline styling (word/s or sentence/s). Proper nesting of inline 
and block elements is essential for valid HTML.
