Q2 Lesson 2: Stylesheets and CSS Syntax

Broswer Output:

    

CSS Syntax Basics:

Uses selectors, properties, and values.
Syntax: selector { property: value; }
Example: h1 { color: blue; font-size: 12px; }

Key Symbols in CSS:

{ } – Curly braces to enclose declarations.
: – Separates property from value.
; – Ends each declaration.

Style Sheet Components:

Selector: Targets HTML elements.
Property: Defines the style attribute (e.g., color, font-size).
Value: Specifies the setting for the property.
Declaration: A single style rule.
Declaration Block: Group of declarations for a selector.

How Style Sheets Work:

Separate content (HTML) from appearance (CSS).
CSS tells the browser how to display HTML elements.

CSS Code Guidelines:

Use <style type="text/css"> for embedded styles.
Enclose CSS in HTML comments () for older browser compatibility (mostly outdated now).

Types of CSS:


External Style Sheet:

Stored in .css files.
Linked using style.css.
Affects multiple pages; easy to maintain.


Embedded (Internal) Style Sheet:

Placed within <style> tags in the  section.
Affects only that specific HTML document.
Overrides external styles.


Inline Style:

Written directly in the HTML tag using the style attribute.
Affects only that specific element.
Overrides both embedded and external styles.


Creating and Linking External CSS:

Write styles in Notepad and save with .css extension.
Link to HTML using <link> tag in the  section.

CSS Comments:

Start with /* and end with */.
Used to explain code and improve readability.
Ignored by browsers.
